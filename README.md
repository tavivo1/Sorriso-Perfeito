## Sobre o Projeto

A **Sorriso Perfeito** é uma clínica odontológica fictícia. Este projeto é uma landing page responsiva e moderna para divulgar serviços, depoimentos de clientes e facilitar o contato com potenciais pacientes. A implementação usa **HTML**, **CSS** e **JavaScript** para proporcionar uma experiência interativa e fluida.

## Visão Geral

### Estrutura do Projeto
- **Home**: Apresentação do consultório e resumo dos serviços.
- **Serviços**: Detalhamento dos principais serviços oferecidos.
- **Galeria de Sorrisos**: Imagens de antes e depois.
- **Depoimentos**: Comentários e avaliações de pacientes fictícios.
- **Contato**: Formulário para facilitar a comunicação com os visitantes.

As tecnologias e ferramentas utilizadas neste projeto são:

## Como executar localmente

1. Clone o repositório:

```bash
git clone <URL-do-repositório>
cd projeto-sorriso-perfeito
```

2. Abra o arquivo `index.html` no seu navegador (duplo clique ou com o botão direito → "Abrir com").

3. Para desenvolvimento, recomendamos usar a extensão Live Server do VS Code para recarregamento automático.
